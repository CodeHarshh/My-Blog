# My-Blog


# 📝 Blog Post CRUD App (Full Stack)

This is a full-stack Blog Post Manager built with **Node.js**, **Express**, and **EJS**. It allows users to **Create**, **Read**, **Update**, and **Delete** blog posts.

---

## 🚀 Features

- View all blog posts
- Create new posts
- Edit existing posts
- Delete posts
- RESTful backend API
- EJS frontend with Axios

---

## 🛠 Tech Stack

- Node.js
- Express.js
- EJS
- Axios
- body-parser
- method-override
